Project Overview

TIDE AI is a mobile productivity application prototype created to explore how digital tools can help users organize tasks, manage schedules, and improve productivity. The project includes both UI/UX design exploration and Flutter application development.

Key Features

📋 Task Management

Create, organize, and manage daily tasks efficiently.

📅 Calendar Management

Plan schedules and manage important activities.

🎯 Productivity Tracking

Track productivity goals and user progress.

🤖 AI Assistant Concept

An AI-inspired assistant interface concept for productivity support.
